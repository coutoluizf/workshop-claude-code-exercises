# Workshop Claude Code — Exercícios

Repositório com os exercícios práticos do workshop de Claude Code.

## Estrutura

| Pasta | Tema |
|-------|------|
| `modulo-01/` | Fundamentos do Claude Code — API REST de tarefas (Hono + Cloudflare Workers) |

## Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/coutoluizf/workshop-claude-code-exercises.git
   cd workshop-claude-code-exercises
   ```

2. Acesse a pasta do módulo desejado e siga o README de cada um:
   ```bash
   cd modulo-01
   npm install
   npm test
   ```

## Pré-requisitos

- [Node.js](https://nodejs.org/) (v18+)
- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) instalado e configurado
