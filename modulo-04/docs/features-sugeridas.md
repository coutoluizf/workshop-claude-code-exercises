# Features sugeridas para o tab-stash

Escolha uma para o Exercício 4 — ou invente a sua.

## Nível 1 — Mais acessíveis

### Tema dark/light manual
Toggle no UI para trocar tema, em vez de só seguir o sistema operacional.

### Contador de abas
Badge no ícone da extensão mostrando quantas abas estão abertas.

### Confirmação ao fechar
Dialog de confirmação antes de fechar tabs que já foram stashed.

## Nível 2 — Intermediárias

### Categorias/pastas
Organizar stashes em coleções (Trabalho, Pessoal, Pesquisa). Criar, renomear, deletar coleções.

### Notas por item
Campo de texto para adicionar observações a cada tab salva. Editável no dashboard.

### Estatísticas
Painel mostrando: tabs salvas no mês, domínios mais frequentes, tempo médio stashed.

## Nível 3 — Desafiadoras

### Sincronização via arquivo
Export/import automático via arquivo JSON para Google Drive, Dropbox, ou pasta local.

### Busca no conteúdo
Buscar não só no título/URL mas no conteúdo da página salva (snapshot do texto).

### Atalhos customizáveis
Interface para o usuário definir seus próprios keyboard shortcuts para stash, open, dashboard.
